# Install

By default, Griffel is a runtime CSS-in-JS engine, you can simply install and use it in code directly:

<Tabs>
<TabItem value="yarn" label="Yarn">

```shell
yarn add @griffel/react
```

</TabItem>
<TabItem value="npm" label="NPM">

```shell
npm install @griffel/react
```

</TabItem>
</Tabs>

For quick start you can use [CodeSandbox template](https://codesandbox.io/s/griffel-react-template-p5e8mh) or check our [try it out feature](/try-it-out).
