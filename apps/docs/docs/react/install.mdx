# Install

By default Griffel contains only runtime version, it's enough to install it and it works:

<Tabs>
<TabItem value="yarn" label="Yarn">

```shell
yarn add @griffel/react
```

</TabItem>
<TabItem value="npm" label="NPM">

```shell
npm install @griffel/react
```

</TabItem>
</Tabs>
